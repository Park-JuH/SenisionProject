# SenisionProject

## 프로젝트 소개
낙상 사고의 위험 요인 제거 서비스 구축

## 개발 기간
2024.04. - 2024.06

## 맴버 구성
1. 이승협(메디컬) : 팀장 및 총괄/메디컬
2. 김희진 : 백엔드
3. 김란 : 디자이너
4. 이윤경 : 풀스택(일단 백엔드 - 안나오면 프론트), 인공지능
5. 박주혁 : 백엔드, 인공지능
6. 서진영 : 메디컬 및 기획
7. 이연희 : 백엔드, 인공지능
8. 박세화 : 프론트 개발
